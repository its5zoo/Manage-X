# Manage X 🚀

Manage X is a modern, web-based management system designed to simplify and organize daily tasks efficiently. It provides a centralized platform to manage services, contacts, bookings, and tickets with a clean, intuitive, and user-friendly interface.

---

## 🛠️ Tech Stack

- **Frontend:** React.js, CSS3 (Modern responsive styling), HTML5
- **Backend:** Node.js, Express.js
- **Version Control:** Git & GitHub

---
